Bio-KMP: 基于双队列验证与自适应跳转的生物KMP算法

📋 项目简介
传统KMP算法在处理基因串联重复序列时易遗漏匹配。本算法引入“记忆队列-候选队列”双缓冲验证机制及自适应跳转策略，确保100%匹配完备性。

🚀 性能优势
 字符比较次数平均减少18.7%
 高密度重复序列处理性能提升39.5%

🖥️ 可视化平台
| 页面 | 功能 |
|------|------|
| 🧬 基因序列分析 | 输入DNA序列、生成典型基因序列 |
| ⚔️ 暴力算法 | 暴力匹配算法演示 |
| 📘 KMP算法 | 传统KMP算法演示 |
| ⚡ 生物KMP算法 | 记忆队列、候选队列演示 |
| 📊 算法对比 | 性能对比图表 |

📦 安装与运行
```bash
git clone https://github.com/1258996/Bio-KMP.git
cd Bio-KMP
python Bio-KMP.py
```

内置案例：点击串联重复按钮 → 分析基因序列

KMP漏检演示：加载KMP漏检案例 → 观察对比

🔬 算法原理
记忆队列：记录跳过的潜在匹配起点
候选队列：缓存部分匹配状态
自适应跳转：动态计算最小安全跳转距离
